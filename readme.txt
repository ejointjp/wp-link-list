WP Link List is shortcode base generate link list plugin.

== Description ==

WP Link List is shortcode base generate link list plugin.

You can choose from 4 types of linked list style.

* Link title Only
* Link title and thumbnail
* Link title and description
* Link title and thumbnail and description

WP Link Listは､ショートコードを使ってリンク集を作成できるプラグインです｡

4種類のリンクリストを作成できます｡

* タイトルのみのリンク
* タイトルとサムネイル
* タイトルと説明文
* タイトルとサムネイルと説明文


== Installation ==

1. Upload `wp-link-list` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can create a Link List by setting page.
